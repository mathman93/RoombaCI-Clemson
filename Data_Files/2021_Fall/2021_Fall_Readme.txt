This folder contains data collected during the Fall 2021 semester.

curvetrackdata.txt: Data collected from Roomba_Cure_Tracking program using a random path
curvetrackdata2.txt: Same as above except for when Roomba is off the path
curvetrackdata3.txt: Same as above except data was taken for different variables
    to show how the Roomba looks when it has moved along a path and what its heading
    is and how it is going towards the seek point
stardata.txt: Data collected from Roomba_Curve_Tracking program using a star path
